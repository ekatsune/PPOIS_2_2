## 3. Описание CLI команд

| Модуль      | Команда              | Аргументы            | Описание                                                       | Пример использования                     | Выход / результат                                           |
|------------|--------------------|-------------------|----------------------------------------------------------------|----------------------------------------|-------------------------------------------------------------|
| **farm**   | `--buycow`          | `<name>`          | Купить корову на ферму                                         | `farm --buycow Мурка`                  | `Корова Мурка куплена`                                     |
|            | `--check`           | —                 | Показать список коров на ферме                                  | `farm --check`                          | Список коров с именами, иными характеристиками                                     |
| **techproc** | `--milking`        | —                 | Подоить всех коров на ферме                                     | `techproc --milking`                     | Все коровы на ферме подоены 1 раз                            |
|            | `--pasteurize`      | `<liters>`        | Сделать пастеризованное молоко                                   | `techproc --pasteurize 10`              | Пастеризовано 10 литров молока, разлито по бутылкам, отправлено на склад                       |
|            | `--yogurt`          | `<liters> <flavour>` | Сделать йогурт с заданным вкусом                                | `techproc --yogurt 5 клубника`        | Произведено 5 литров клубничного йогурта, разлит по стаканчикам, отправлен на склад             |
| **workshop** | `--check`          | —                 | Проверить количество молока в цеху                               | `workshop --check`                       | Вывод объема молока в цеху                                  |
| **warehouse** | `--check`         | —                 | Проверить продукцию на складе                                    | `warehouse --check`                      | Список продукции на складе                                   |
| **customer** | `--create`         | `<name>`          | Создать клиента                                                | `customer --create Almi`                 | `Клиент Almi создан`                                        |
|            | `--check`           | —                 | Показать всех клиентов                                         | `customer --check`                       | Список клиентов                                            |
|            | `--order`           | `<name>`          | Сделать заказ от клиента                                        | `customer --order Almi`                  | CLI запрос на добавление товаров (milk/yogurt)             |
| **orders**  | `--check`           | —                 | Просмотр очереди заказов                                        | `orders --check`                         | Список заказов с деталями                                   |
| **player**  | `--take`            | —                 | Выполнить первый заказ в очереди                                 | `player --take`                          | Заказ выполнен               |

**Примечания:**
- Для создания йогурта сначала нужно подоить коров.  
- При добавлении заказа йогурта принимает формат:  

`yogurt <quantity> <flavour>`

- При добавлении заказа молока:  

`milk <quantity>`
